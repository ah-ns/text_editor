## Text Editor Plan

### What it needs to do:

+ Open a file, or create one if it doesn't exist
+ Exit the document
+ Save the document
+ Display the contents of the file on screen
+ Write things and backspace
+ Scroll the document
+ Have a cursor

### Other things (Stretch goals):

+ Line numbers
+ Highlight current line
+ Zoom in/out
+ Commands like undo and redo
+ Syntax highlighting
